Dynamic Background Color

#3498db

This page's background changes based on the URL parameter.

Important: When using hex codes with a # symbol, you need to URL encode it as %23.

Example: ?c=%237e3f14 instead of ?c=#7e3f14

How to use:

Add a color parameter to the URL in one of these formats:

  • Hex code (without #): ?c=7e3f14
  • Hex code (with encoded #): ?c=%237e3f14
  • Color name: ?c=blue or ?c=darkcyan

Try these examples:

?c=7e3f14 ?c=%237e3f14 ?c=ff5733 ?c=0f0 ?c=blue ?c=darkcyan

OBS Users

To show only the background in OBS (hide this content panel):

Add this CSS override in your browser source settings:

.container { display: none; }
